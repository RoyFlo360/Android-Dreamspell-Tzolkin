# mayan_magic_wisdom_palette

Chosen color palette for the Tzolkin navigation redesign (and app theming going forward).

| Role | Name | Hex |
|------|------|-----|
| Primary base — screens, nav bar | Indigo obsidian | `#191428` |
| Secondary — headers, active states | Amethyst | `#4B2E83` |
| Content blocks / cards (sacred green) | Jade | `#1F5C45` |
| Cool secondary accent — cards, informational | Turquoise | `#2A9D8F` |
| Single accent — highlights, active icons, key numbers (maize gold) | Mystic gold | `#C9A227` |
| Warm depth tone — cards, dividers | Terracotta | `#B5451E` |
| Light neutral — text-on-dark, light-mode bg | Warm cream | `#F2E8D5` |

Mockups: `scratchpad/nav_concepts_mayan.html`.
